# cQQcVoice

#### 介绍
cQQcVoice(customQQchatVoice)是一个可以自定义发送的QQ聊天语音的软件    
注意: 发出的语音只能被安卓端接收    
系统要求: >= Android 5.0

#### 软件架构
ARM

#### 安装教程
自己编译或[下载](https://gitee.com/zhou2008/cQQcVoice/releases)编译好的版本    
[使用教程](https://gitee.com/zhou2008/cQQcVoice/raw/master/1.gif)
